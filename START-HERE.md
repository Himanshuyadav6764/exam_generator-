## All One-Line Commands:

| Command       | What Happens                              |
| ------------- | ----------------------------------------- |
| `.\dev.bat`   | Starts Backend + Frontend + Opens Browser |
| `.\start.bat` | Same as above                             |
| `.\start.ps1` | PowerShell version                        |
| `npm start`   | npm version                               |

---

## Step-by-Step:

1. **Open PowerShell or CMD**
2. **Type:**
   ```bash
   cd C:\LOGIN_PAGE
   .\dev.bat
   ```
3. **Wait 15 seconds**
4. **Browser opens automatically!**
